Usage Guidelines

Rules for consistent, legible typography across the Margin platform.

Tabular Figures

All financial data uses font-variant-numeric: tabular-nums lining-nums. Proportional figures break column alignment and cause layout shift on live data.

KPI De-emphasis

In hero KPI displays, the currency symbol renders at 55% of the main figure and cents at 50%. The whole-dollar figure is what matters.

Minimum Body Size

Body text must never be smaller than 16px. Table data may go to 14px. Labels may go to 11px but never smaller.

Numeric Alignment

Right-align all numeric columns. Pad with trailing zeros: $829,892.00 not $829,892.

Color for Polarity

Positive values use Violet 600. Negative values use Scarlet 500. Always pair with +/− signs — never rely on color alone.

Label Tracking

Uppercase labels use letter-spacing: 0.08em at weight 500. This prevents the visual clumping uppercase creates at small sizes.
