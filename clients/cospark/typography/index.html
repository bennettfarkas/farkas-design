Usage Guidelines

Rules for consistent, legible typography across screen and print.

Minimum Body Size

Body text must never be set smaller than 16px (1rem) on screen or 10pt in print. Anything below this threshold degrades readability.

Headline Line Height

Headlines use a tight line height of 0.9–1.2 to create visual density and impact. The larger the type, the tighter the leading.

Body Line Height

Body copy requires generous leading of 1.5–1.6 for comfortable sustained reading. Keep line length to 60–70 characters.

Financial Figures

All financial data uses DM Sans Bold 700 with tabular numerals (font-variant-numeric: tabular-nums) for clean column alignment.

Legal Minimum

Legal and regulatory text may be set at the Caption level — 0.75rem / DM Sans Medium 500, uppercase — but never smaller.

Measure

Body text blocks should be constrained to 60–70 characters per line. Wider measures fatigue the eye and reduce comprehension.
